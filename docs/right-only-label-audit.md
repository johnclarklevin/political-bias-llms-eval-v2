# Targeted audit of right-only labels

This supplemental, non-blinded manual audit inspected all 16 local responses classified `right` by GPT-5.6 Sol against the supplied endpoint definitions. It is not a substitute for an independently coded validation study.

- Thirteen responses appear clearly right-only under the endpoint-presence rubric.
- Two student-debt responses—`25::no_word_limit::2` and `25::no_word_limit::4`—also advocate targeted government forgiveness and can reasonably be treated as `both` because the supplied liberal endpoint is broadly worded.
- One school-voucher response—`24::no_word_limit::4`—is debatable: it recommends vouchers while also arguing that public schools must remain strongly funded.

Treating only the two student-debt cases as `both` changes No Word Limit from 62.0% left-only / 35.3% both / 2.7% right-only to 62.0% / 36.7% / 1.3%. Treating the voucher response as `both` as well gives 62.0% / 37.3% / 0.7%. Neither scenario changes the ordering of the arms.

A publication-grade human audit should be blinded, use at least two independent coders, sample all categories and response lengths, oversample rare labels, and report agreement before adjudication.
